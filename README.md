# KeenKeeper — Keep Your Friendships Alive

> **KeenKeeper** is a web app that allows any user to connect with people they know as friends. This application built with React. See All Friends, track your notification , and also track your calls,texts & video calling with a view of pie charts — all in one beautifully designed platform.

## Technology Used

- **React.Js**
- **React Router DOM**
- **TailwindCss**
- **Daisy UI**
- **Recharts**
- **React spinners**
- **React Toastify**
- **React Icons**

## Features

- **All Friends & Details** - Browse your all friends in your friends list. when clicked on a friends card it's redirect the details page and show the friends details.This details page have three workable button called , call, text & video.When an user click on this it's show up an success messages with toastify & refer it into the history section.

- **Communication History with Friends** - User can track all the history into history section.By default an user see "No history are available right now!" if the history is empty.If the history section have histories data, user can filter them by their three types of filtering value called call, text & video & also track them with the date. And one more thing user can also see their histories data in a view of pie charts which is allocated in the stats section.

- **Not Found Page** - If an user redirect an unknown path or invalid path , then show an error page with an error message called "404 Not Found", and user also have to see a "Back to Home" button for redirect their path to the home.
