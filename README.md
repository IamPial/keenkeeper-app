# KeenKeeper — Keep Your Friendships Alive

> **KeenKeeper** is a web app that allows any user to connect with people they know as friends. This application built with React. See All Friends, track your notification , and also track your calls,texts & video calling with a view of pie charts — all in one beautifully designed platform.

## Technology Used

**React.Js**
**React Router DOM**
**TailwindCss**
**Daisy UI**
**Recharts**
**React spinners**
**React Toastify**
**React Icons**
